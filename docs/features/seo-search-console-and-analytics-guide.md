# 크리에이박스 검색 최적화(SEO) 및 구글 애널리틱스(GA4) 자동화 연동 가이드

본 문서는 크리에이박스(CreAibox) 플랫폼이 제공하는 **초간편 검색 엔진 노출(SEO) 시스템**과 **구글 애널리틱스(GA4) 무설정 자동 발급 기술**에 관한 교육 및 마케팅(홍보)용 안내문입니다.

---

## 📑 1. 구글 / 네이버 검색 노출 (서치콘솔 연동) 가이드

사용자가 크리에이박스에서 어떤 주소(도메인) 형태로 블로그를 개설하느냐에 따라 검색엔진 최적화 방식이 다릅니다.

### 1.1. 기본 서브도메인 사용자 (`아이디.creaibox.com`)
* **핵심 가치**: **"아무런 셋팅 없이 글만 쓰면 구글/네이버에 자동으로 검색 노출이 개시됩니다!"**
* **작동 원리**:
  * 최상위 대표 도메인인 `creaibox.com`은 이미 크리에이박스 본사 측에서 구글 서치콘솔 및 네이버 서치어드바이저에 정식 등록을 완료했습니다.
  * 검색엔진의 크롤링 로봇은 `creaibox.com` 산하의 모든 서브도메인을 대표 사이트의 일부(Subdomain)로 취급하여 자동으로 접근 및 기사 색인(Index) 작업을 진행합니다.
  * 따라서 기본 주소 사용자는 복잡한 서치콘솔 가입이나 도메인 소유권 확인 절차를 밟을 필요가 전혀 없습니다.

### 1.2. 개인 독립 도메인 사용자 (`본인개인도메인.com` 등)
* **핵심 가치**: **"발급받은 인증키만 크리에이박스 관리자에 복사-붙여넣기 하세요. 코딩 없는 프로 SEO 환경이 완성됩니다."**
* **작동 원리**:
  * 독립 도메인은 사용자의 개별 소유 자산이므로 검색포털에 최초 1회 "도메인 소유권 확인(인증)" 단계를 반드시 거쳐야 합니다.
* **연동 절차**:
  1. 네이버 서치어드바이저 / 구글 서치콘솔에 로그인하여 본인의 독립 도메인 주소를 추가합니다.
  2. 시스템이 제공하는 **[HTML 태그(메타태그)]** 소유권 확인 방식을 선택한 후, 제공되는 인증키(예: `google-site-verification` 또는 `naver-site-verification`)를 복사합니다.
  3. 크리에이박스 **[공식 블로그 관리 > SEO 및 연동 관리]** 화면에 복사한 키를 붙여넣고 저장합니다.
  4. 웹마스터 도구 화면에 본인 도메인의 사이트맵 주소(`https://도메인/sitemap.xml`)와 RSS 피드 주소(`https://도메인/feed`)를 딱 1회 제출하여 등록을 완마침합니다.
  5. 이후부터는 사용자가 글을 쓸 때마다 시스템이 사이트맵과 피드를 자동으로 업데이트하여 검색엔진에 무인 전송합니다.

## 📊 2. 구글 애널리틱스 (GA4) 100% 무인 자동 발급 시스템

일반적으로 방문자 데이터를 추적하기 위해 구글 애널리틱스에 가입하고, 스트림을 개설하여, 복잡한 추적 코드(Global Site Tag)를 복사해서 홈페이지 HTML의 `<head>` 태그 사이에 직접 코딩해 넣는 고난이도 작업을 수행해야 합니다. 

크리에이박스는 이 모든 과정을 **완전 무인 자동화(Zero-Setup Automation)**했습니다.

```
[동작 프로세스]
사용자 브랜드 도메인 승인 (어드민)
       │
       ▼
Google Analytics Admin API 호출 (백엔드 서버)
       │
       ▼
해당 도메인 전용 GA4 웹 데이터 스트림 및 측정 ID (G-XXXXXX) 즉시 자동 개설
       │
       ▼
사용자 프로필 설정(profile.extra_configs)에 자동 저장 및 사이트 헤더 코드 자동 주입
       │
       ▼
사용자 가입/추가 설정 일절 없이 실시간 통계 추적 즉시 시작!
```

### 2.1. 사용자 편의성 극대화 (No-Code, No-Setup)
* 사용자는 구글 애널리틱스 사이트에 가입할 필요조차 없습니다.
* 크리에이박스 블로그가 개설되는 즉시 백엔드 엔진이 구글 서버와 통신하여 해당 블로그만을 위한 데이터 분석 채널을 즉시 열어주며, 추적 스크립트도 알아서 안전하게 탑재해 줍니다.

### 2.2. 소유권 인증과의 시너지 효과 (프리패스 통과)
* 구글 애널리틱스 코드가 사이트에 자동으로 심어지기 때문에, 사용자가 나중에 구글 서치콘솔에 해당 도메인을 등록할 때 **"이미 구글 애널리틱스 추적 태그가 감지되어 소유권이 즉시 확인되었습니다!"**라며 번거로운 파일 업로드나 TXT 레코드 설정 없이 원클릭 프리패스로 인증이 통과됩니다.
