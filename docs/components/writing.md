# 블로그 글쓰기 컴포넌트 (Writing Studio Components)

블로그 콘텐츠 생산 모듈의 핵심 리치 텍스트 에디터 및 채널별 연동 컴포넌트 상세 명세입니다.

---

## 1. 만능 블로그 에디터 (`src/components/writing/editor/UniversalBlogEditor.tsx`)

블로그 도메인에 관계없이 사용되는 통합 에디터 컴포넌트입니다.
* **Tiptap 기반 구성**: 볼드, 이탤릭, 링크, 헤더 계층, 유튜브 삽입, 정적 테이블, 정렬 기능 등을 통합 툴바와 매핑.
* **스마트 미디어 업로드**:
  - 드래그 앤 드롭 및 클립보드 붙여넣기(`Paste`) 시 이미지 바이너리를 가로채 Supabase Storage(`generated-images`)에 비동기 업로드하고 에디터에 공개 이미지 URL 주입.
  - 임베디드 이미지에 캡션(Caption)과 ALT 설명 태그를 지정하는 패널 결합.
* **영속화 캐시 (Auto-Save)**: 타이핑 멈춤 시 Zustand 스토어와 연동해 로컬 세션에 작성 상태를 백업하여 불의의 브라우저 종료 시 임시 본문 유실을 방지.
* **AI 포스팅 재창조 (Spin-Rewriting)**: 타겟 글 주소(네이버 블로그 URL)를 입력하면 본문 데이터를 추출하여 에디터에 로드하고, 핵심 키워드/주제/문맥을 기반으로 유사성 우회 로봇을 적용해 새로운 오리지널 콘텐츠로 전면 재창조하여 에디터에 즉시 렌더링하는 기능 탑재.

## 2. 네이버 포스팅 컴포넌트

### 2-1. `NaverAnalysisTower`
* **역할**: 네이버 노출용 키워드 난이도와 경쟁 강도를 검사하는 패널.
* **기능**: 포스트 제목/본문 글자 수 지표, 권장 이미지 수 대조 검사기 제공.

### 2-2. `NaverHotKeywordMatrix`
* **역할**: 네이버 실시간 검색 추이 및 키워드 순위 매트릭스 차트.
* **기능**: 관련 카테고리의 인기 해시태그 및 유도 태그 제안.

### 2-3. `NaverRecreateTab`
* **역할**: 경쟁사 글의 URL 또는 임의의 문장을 복사해 붙여넣으면, AI 분석기가 문체와 어투를 바꿔서 저작권 유사성을 회피한 고품질 변형 글을 생산하는 패널.

## 3. 워드프레스 특화 컴포넌트 (`src/components/writing/wp/tabs/`)

워드프레스 다중 독립 블로그 연동 및 자동 업로드를 위한 서브 패널 모음입니다.
* **`ApiConfigTab`**: 사용자가 본인의 워드프레스 도메인 주소, REST API 아이디, 애플리케이션 비밀번호(App Password)를 등록해 연결을 진단.
* **`PublishTab`**: 본문에 연계된 이미지들과 글을 한꺼번에 JSON 페이로드로 직렬화하여 워드프레스 REST API(`/wp-json/wp/v2/posts`)에 직접 POST 통신을 수행해 퍼블리싱.
* **`ThumbnailTab`**: AI 이미지 스튜디오에서 생성한 썸네일을 워드프레스 미디어 라이브러리에 선제 업로드한 후 글의 '특성 이미지(Featured Image)' ID로 설정 연동.

## 4. CreAibox 독자 포스팅 컴포넌트 (`src/components/writing/creaibox/tabs/`)

독자 블로그에 올리기 전 구글 검색 크롤러 노출 최적화를 위한 보조 도구들입니다.
* **`CreaiboxSchemaPanel`**: 구글 구조화 데이터 리치 스니펫(FAQPage, Article schema 등)을 에디터 출력용 JSON-LD 스키마 마크업으로 출력해 주는 입력 폼.
* **`CreaiboxSeoOptimizationPanel`**: 포커스 키워드 빈도 비율, 헤더 구조(`h1`~`h3` 규칙), 이미지 ALT 속성 존재 여부, 메타 설명글 길이를 실시간 검사해 검색 엔진 친화 점수 계산.
