# 콘텐츠 아이디어 허브 확장 분류 맵 (10대분류 / 50상세분야 제안)

현재의 7대분류 / 35상세분야 체계를 논리적으로 확장하고 보강하여 구성한 **10대분류 / 50상세분야** 최종 개편안입니다. 
가로 5열 그리드 레이아웃에서 빈 슬롯 없이 완벽하게 대칭되는 10개 행(총 50개)을 형성하도록 상세 분야 개수를 조정(15개 신설)했습니다.
