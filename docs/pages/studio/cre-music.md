# Cre Music 플레이어 페이지 명세 (Page Specification)

## 1. 개요 (Overview)
* **페이지 경로**: `/studio/music/cre-music` (실제 파일: `src/app/studio/music/cre-music/page.tsx`)
* **목적**: 관리자가 업로드해 둔 보컬 트랜스 앨범(`Awakening`)의 트랙 목록을 구글 드라이브 지정 폴더로부터 실시간 스캔하여 스포티파이 스타일의 인터페이스 상에서 동적으로 감상할 수 있는 플레이어 화면.
* **접근 대상**: 로그인한 사용자 전체 (단, 구글 드라이브 음원 목록 조회 API는 `ALLOWED_TESTER_EMAILS` 화이트리스트에 있는 사용자만 가능하며, 그 외 사용자는 데모 음원으로 자동 전환되어 체험 가능).

---

## 2. 화면 구성 및 UI 스펙 (Layout & UI Specs)

### 2-1. 좌측 사이드바 (Sidebar Panel)
* **네비게이션 메뉴**:
  * **뮤직 스튜디오 홈**: `/studio/music`으로 바로 이동할 수 있는 이전 단축 링크 버튼 배치.
  * **Cre Music 로고**: 회전하는 레코드 아이콘(`Disc3`)과 화려한 브랜드 네임 렌더링.
  * **보컬 트랜스 라이브러리**: 현재 액티브 상태를 표시하는 전용 네비게이션 아이템.
  * **Awakening 앨범 플레이리스트 단축키**: 앨범에 빠르게 초점을 맞출 수 있는 링크 블록.

### 2-2. 메인 뷰어 영역 (Main Content Panel)
* **상단 헤더바 (Header)**:
  * **상태 표시**: "Google Drive 20TB 연동 음악 플레이어" 및 스파클 아이콘 탑재.
  * **동기화 버튼**: 클릭 시 드라이브 API(`/api/music-studio/list`)를 호출하여 음원 목록을 새로고침하는 기능.
  * **드라이브 열기**: 구글 드라이브로 원격 이동할 수 있는 링크 버튼.
* **앨범 배너 (Album Banner)**:
  * 장르 정보 (`ALBUM / Vocal Trance`), 앨범 제목(`Awakening`), 아티스트 정보(`CreAIbox Studio`), 트랙 수, 그리고 구글 드라이브 연동 상태 뱃지 표시.
  * 퍼플-블루 계열의 그라데이션 커버 아트 구성.
* **음원 목록 테이블 (Track Table)**:
  * 각 열 정보: 곡 번호, 곡 제목(확장자가 파싱된 제목 및 원본 파일명 서브타이틀), 파일 크기, 생성 일자, 재생 컨트롤 버튼.
  * 재생 중인 곡은 텍스트 및 번호 색상이 민트/시안 계열로 강조되며, 순번 영역에 실시간 오디오 바운스 바 이펙트(Bounce Bar Animation)가 노출됩니다.

### 2-3. 글로벌 하단 플레이 바 (Global Play Bar)
* **현재 곡 상세정보**: 재생 중인 곡의 앨범 아트 썸네일, 제목, 앨범 명 노출.
* **음악 재생 제어 영역 (Play Controls)**:
  * 셔플 버튼(토글 시 민트 강조), 이전 곡 버튼, 재생/일시정지 버튼, 다음 곡 버튼, 루프 한곡 반복 버튼(토글 시 민트 강조).
  * HTML5 `<audio>` 노드의 상태와 양방향 동기화되는 실시간 진행도 바(Seek Bar) 배치. 진행바 클릭 시 해당 위치로 탐색 재생(Seeking)을 수행합니다.
* **우측 볼륨 컨트롤 영역**:
  * 음소거 단추(`Volume2` ➡️ `VolumeX`) 및 볼륨 조정을 위한 0 ~ 1 범위의 range 입력 슬라이더 제공.

## 3. 핵심 사용자 흐름 (User Flow)
1. 사용자가 뮤직 스튜디오 홈 페이지(`/studio/music`)의 메뉴 카드 또는 빠른 실행 링크를 통해 `Cre Music 플레이어` 페이지로 진입합니다.
2. 페이지 진입 즉시 백엔드 `/api/music-studio/list`에 조회를 요청합니다.
3. **권한 확인 성공**: 사용자가 화이트리스트에 있고 정상 로그인된 경우, 구글 드라이브의 실제 오디오 파일 목록이 플레이어 리스트에 채워집니다.
4. **권한 확인 실패 / 연동 오류**: 에러 메시지가 상단에 알림판 형태로 노출되며, 플레이어는 에러로 멈추지 않고 시스템에 내장된 3개의 데모 음원(`SoundHelix` CDN 등)을 목록으로 로드해 정상 플레이 체험을 보장합니다.
5. 임의의 트랙을 선택해 재생 버튼을 클릭하면, 내부 백엔드 스트리밍 프록시 API(`/api/music-studio/stream?id=[id]`)를 매개로 구글 드라이브 오디오 바이너리를 `Range` 헤더와 함께 206 Partial Content 스트림으로 전송하여 즉시 재생됩니다.
6. 곡이 종료되면 다음 곡이 자동 재생(Auto-Play Next)되며, 셔플 또는 루프 설정에 따라 재생 대상이 바뀝니다.
