# AI 홈페이지 제작 스튜디오 페이지 명세서 (Page Specification)

본 문서는 CreAIbox 플랫폼 내 **"AI 홈페이지 제작 스튜디오"** 관련 페이지들의 경로 설계, UI 컴포넌트 배치, 비즈니스 규칙 및 클라이언트 인터랙션 흐름을 설명하는 공식 페이지 명세서입니다.

---

## 1. 개요 및 경로 설계 (Routing Overview)
본 스튜디오는 사용자의 홈페이지 상태와 비즈니스 요금제에 따라 여러 하위 탭으로 분할 노출되는 다중 경로 구조로 설계되었습니다.

| 메뉴 명칭 | URL 경로 | 주요 역할 | 노출 대상 |
| :--- | :--- | :--- | :--- |
| **스튜디오 홈** | `/studio/client-site-builder` | At a Glance 요약 대시보드, 유입 문의 요약, 가이드 | Business 이상 |
| **AI 홈페이지 빌더** | `/studio/client-site-builder/builder` | 신규 마법사(Wizard) 또는 실시간 섹션 편집기(Editor) | Business 이상 |
| **디자인 테마 라이브러리**| `/studio/client-site-builder/themes` | 워드프레스식 카테고리별 테마 목록 탐색 및 원클릭 적용 | Business 이상 |
| **고객 문의 관리** | `/studio/client-site-builder/inquiries` | 유입된 견적/상담 상담 신청 내역 처리 및 메모 보관 | Business 이상 |
| **페이지 & 글 관리** | `/studio/client-site-builder/posts` | 고정 정적 페이지 바로가기 및 공지/블로그 포스트 CRUD | Business 이상 |
| **홈페이지 설정** | `/studio/client-site-builder/settings` | 대표자명, 주소, 연락처, SNS 링크 및 GA4 연동 설정 | Business 이상 |

## 2. 레이아웃 셸 구성 (`layout.tsx`)
모든 하위 경로는 동일한 글로벌 껍데기 레이아웃(GNB Header)과 서브 탭바를 공유하여 빠른 탐색과 즉시 렌더링(Loading-Free)을 보장합니다.

### 2-1. GNB 헤더 영역
1. **운영 정보 노출**:
   - 활성 홈페이지가 존재할 경우, `[운영 중]` 그린 뱃지와 서브도메인 브랜드명(`brand_id.creaibox.com`)을 상시 노출합니다.
   - 우측에 **"내 홈페이지 바로가기"** 버튼(새 창 열기)을 노출하여 실시간 배포 상태를 바로 확인할 수 있습니다.
2. **다중 사이트 스위처 (Multi-site Switcher)**:
   - 사용자가 소유한 홈페이지가 2개 이상일 경우, 헤더 내에 셀렉트 박스 형태로 드롭다운을 제공합니다.
   - 드롭다운 값을 조작하면 `SiteBuilderProvider` 컨텍스트의 `selectedSite`가 변경되고, 이에 맞춰 React 컴포넌트의 `key` 속성이 새로고침되어 하위 페이지들이 깜빡임 없이 즉각 재렌더링됩니다.
3. **새 홈페이지 추가 버튼**:
   - 언제든지 "➕ 새 홈페이지 추가" 버튼을 누르면 전역 `isCreatingNewSite`가 `true`로 전환되며 즉시 마법사(`/builder`)로 리다이렉트 처리됩니다.

### 2-2. 서브 탭바 영역
- GNB 헤더 바로 아래에 6대 핵심 메뉴 탭바를 고정 렌더링합니다.
- Next.js의 `usePathname`을 활용해 현재 브라우저 URL과 100% 매칭되는 탭을 HSL tailored 컬러(`text-emerald-400` 및 `border-emerald-500` 등)로 굵고 선명하게 하이라이트 처리합니다.

## 3. 페이지별 상세 구성 명세

### 3-1. 스튜디오 홈 (`/studio/client-site-builder`)
- **홈페이지 무보유 회원**:
  - 대형 그래디언트 배너 카드로 환영 인사를 건네며 "AI 홈페이지 빌더 시작하기" 및 "디자인 테마 먼저 구경하기" 이동 버튼 제공.
- **홈페이지 보유 회원**:
  - **At a Glance**: 활성 테마명, 최초 개설일, 운영 도메인 주소, 활성 섹션 수, 블로그 글 수, 누적 상담 신청 건수를 한눈에 요약 노출.
  - **최근 고객 문의**: 최근 유입된 3건의 상담 신청 내역을 타임라인 테이블로 노출하고, 상세 조회를 위해 "전체 보기" 링크 제공.
  - **핵심 기능 단축 카드**: 빌더, 테마 스토어, 글 관리, 마스터 설정으로 다이렉트 랜딩되는 4개 프리미엄 카드 그리드 탑재.

### 3-2. AI 홈페이지 빌더 (`/studio/client-site-builder/builder`)
- **생성 모드 (Wizard)**:
  - `sites.length === 0`이거나 `isCreatingNewSite === true`일 때 작동.
  - 사용자는 브랜드 ID, 회사명, 크롤링할 블로그 주소를 입력합니다.
  - **테마 자동 바인딩**: 테마 스토어 등에서 `?theme=restaurant_warm`과 같은 쿼리 스트링을 달고 진입했을 경우, 1단계 AI 기획 생성 API 전송 시 해당 템플릿에 특화된 컨셉으로 섹션 문구를 초안 설계하도록 시스템 프롬프트 가이드가 자동 탑재되고 템플릿이 사전 선택 고정됩니다.
- **편집 모드 (SectionEditor)**:
  - 기존 홈페이지를 운영 중일 때 작동.
  - 실시간으로 좌측 열에서 메인 섹션들의 제목/설명을 타이핑 수정 및 삭제하고, 우측 sticky 사이드바를 통해 추천 매칭 섹션을 추가/조정할 수 있는 2단 고해상도 비주얼 편집 화면을 제공합니다.

### 3-3. 디자인 테마 라이브러리 (`/studio/client-site-builder/themes`)
- **카테고리 필터링**: "All", "Business", "Education", "Food", "Portfolio" 등 5대 카테고리 탭 버튼을 클릭하면 템플릿 레지스트리의 목록이 필터링 렌더링됩니다.
- **테마 카드 호버 이펙트**: 마우스 호버 시 썸네일 이미지가 부드럽게 확대되며 줌인 쉐도우 효과와 함께 "테마 상세 보기" 뱃지가 노출됩니다.
- **상세 보기 모달창**:
  - **디자인 상세 명세**: 테마가 요구하는 구글 폰트명, 모서리 라운딩 크기, 테마 컬러 6색(Primary, Secondary 등)을 컬러 칩 서클 배포로 시각화하여 사용자가 한눈에 스타일 정체성을 파악할 수 있도록 돕습니다.
  - **이 테마로 만들기**: 신규 사이트 제작 파라미터를 들고 빌더로 리다이렉트합니다.
  - **이 테마로 내 홈페이지 변경**: 운영 중인 홈페이지가 있을 경우, 모달 내에서 클릭 한 번으로 활성 사이트의 템플릿 테마를 DB 상에서 즉시 교체 및 배포 반영합니다.

### 3-4. 페이지 & 글 관리 (`/studio/client-site-builder/posts`)
- **1 탭 (페이지 관리)**:
  - 메인 홈(`/`), 회사소개(`/about`), 서비스안내(`/services`), 문의하기(`/contact`)의 발행 현황 및 바로가기 링크 제공.
- **2 탭 (공지 및 블로그)**:
  - `site_posts` 테이블을 직접 조회하여 `notice`(공지) 및 `board`(일반 블로그) 글을 리스트업하고 관리합니다.
  - **➕ 새 글 작성하기**: 제목, 구분, 본문 작성 에디터 폼 모달을 제공하여 원클릭 발행을 구현합니다.
  - **공지사항 상단 고정**: `is_pinned` 값을 업데이트하여 최우선 공지를 고정하고 핀 아이콘 컬러를 활성화합니다.
  - **영구 삭제**: 휴지통 아이콘 클릭 시 사용자 재확인 후 DB 데이터를 즉시 영구 삭제하여 실시간 홈페이지 피드에서 제외합니다.

### 3-5. 고객 문의 관리 및 마스터 설정
- **고객 문의 관리 (`/inquiries`)**:
  - 홈페이지 상담 신청 폼을 통해 인입된 모든 방문자 리드 목록을 테이블로 보여주고, 상세 상담 내용 확인, 처리 상태 변경(`접수 / 상담중 / 완료`), 관리자 전용 상담 내부 메모를 저장 및 영속화합니다.
- **홈페이지 설정 (`/settings`)**:
  - 공식 기업명, 연락처, 주소 및 대표자 성명, 사업자번호 등 마스터 정보 변경 폼을 제공합니다.
  - 네이버 블로그, 인스타그램, 유튜브 채널 링크와 구글 애널리틱스(GA4) 추적 측정 ID(`G-XXXXX`)를 입력받아 실시간 홈페이지 소스 코드에 주입 배포합니다.
