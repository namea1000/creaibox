# 키워드 트렌드 스튜디오 화면 정의서 (Page Specification)

이 문서는 키워드 트렌드 분석 스튜디오 및 서브 도구 화면들의 라우팅 주소와 화면 구성을 기록한 사양서입니다.

---

## 1. 메인 홈 화면
* **경로**: `/studio/keyword` -> `src/app/studio/keyword/page.tsx`
* **설명**: 키워드 연구소 메인 제어 센터. 전체 추적 통계 요약 카드, 10대 핵심 분석 도구 네비게이션 그리드, 최근 분석 히스토리 및 빠른 검색어 클라우드를 구성합니다.

---

## 2. 동적 상세 분석 도구
* **경로**: `/studio/keyword/[section]` -> `src/app/studio/keyword/[section]/page.tsx`
* **설명**: URL의 `[section]` 세그먼트 매개변수를 판단하여 다음 컴포넌트를 마운트하고 뒤로 가기 링크를 포함한 브레드크럼 헤더 레이아웃을 제공합니다.

### 2-1. 키워드 대량 조회
* **세그먼트**: `/studio/keyword/bulk`
* **기능**: 여러 키워드를 줄바꿈 또는 콤마로 입력 후 동시 분석하여 엑셀 다운로드 포맷으로 노출.

### 2-2. 연관 키워드 발굴
* **세그먼트**: `/studio/keyword/related`
* **기능**: 시드 키워드 검색 시 포털 연관어 검색량, PC/Mobile 점유율, CPC 및 경쟁 강도 데이터 매칭.

### 2-3. 형태소 분석기
* **세그먼트**: `/studio/keyword/morphology`
* **기능**: 본문 텍스트 내 형태소 분석을 수행하여 명사 빈도수, 품사 분포 시각화 및 키워드 최적 조합 제안.

### 2-4. 실시간 순위 추적
* **세그먼트**: `/studio/keyword/rank`
* **기능**: 타겟 검색 포털(Naver, Google, Youtube)과 추적할 URL을 바인딩하여 랭킹 변동 실시간 트랙.

### 2-5. 트렌드 급상승 분석
* **세그먼트**: `/studio/keyword/rising`
* **기능**: 상승 속도가 높은 트렌디한 관심 검색어를 차트 스파크라인과 함께 실시간 피드로 나열.

### 2-6. 유튜브 키워드 분석
* **세그먼트**: `/studio/keyword/youtube`
* **기능**: 동영상/Shorts 별 시청 예측 클릭률(CTR) 및 최적화 추천 태그 생성 클립보드 복사 지원.

### 2-7. SEO 경쟁 분석
* **세그먼트**: `/studio/keyword/seo`
* **기능**: 경쟁사 상위 노출 콘텐츠와 대비하여 권장 글자 수, 이미지 수 및 최적화 SEO 체크리스트 구성.

### 2-8. AI 키워드 전략 생성
* **세그먼트**: `/studio/keyword/strategy`
* **기능**: 타겟 타겟층과 어조 옵션을 반영하여 AI 제목, 개요, 태그 기획서 원클릭 생성.

### 2-9. 자동 콘텐츠 연결
* **세그먼트**: `/studio/keyword/workflow`
* **기능**: 한 번의 키워드 입력으로 통계 수집과 AI 전략서 생성을 마친 뒤, 글쓰기/영상 에디터로 데이터를 연동하는 마법사 뷰.

### 2-10. 트렌드 대시보드
* **세그먼트**: `/studio/keyword/dashboard`
* **기능**: 6개월 누적 트렌드 지수(SVG 영역 차트), 유입 채널 점유율 및 맞춤 추천 키워드 모니터링 보드.
