# 스튜디오 Tools (Tools Studio)

이 문서는 스튜디오 Tools 모듈의 현재 아키텍처 및 세부 유틸리티들에 대해 설명하는 상시 운영용 문서입니다.

---

## 1. 목적 (Purpose)
* 이미지 가공, 프롬프트 튜닝, 문서 요약, 코드 정돈, 해시태그 매핑 등 콘텐츠 제작 과정 전반에 걸쳐 기획자와 제작자가 반복적으로 실행해야 하는 보조 성격의 미니 유틸리티 도구들을 한자리에 집약하여 제공합니다.

---

## 2. 주요 기능 (Main Features)
1. **AI 누끼 제거 (`bg-remover`)**: 드래그앤드롭 이미지에서 투명 배경화한 PNG 이미지 추출 시뮬레이션.
2. **PDF 문서 분석 (`pdf-analyzer`)**: PDF 문서의 내용 요약, 키워드 추출 및 문서 내용을 질문하고 응답받는 Doc Q&A 채팅 보드.
3. **AI OCR 문자 추출 (`ocr`)**: 업로드한 인쇄/자필 스캔본 내 문자 영역 판독 및 텍스트 전환 복사 기능.
4. **AI 프롬프트 개선기 (`prompt-enhancer`)**: 키워드 초안을 미드저니, 수노 모델에 최적화된 영문 고급 프롬프트 코드로 확장.
5. **AI 프롬프트 번역기 (`prompt-translator`)**: 한글 묘사를 영문 키워드 태그 묶음 또는 영문 문장으로 정밀 번역.
6. **해시태그 생성기 (`hashtag`)**: 플랫폼별(인스타그램, 유튜브, 블로그 등) 대형/세부 타겟 해시태그 생성 및 복사.
7. **유튜브 썸네일 다운로더 (`youtube-thumbnail`)**: 유튜브 URL 입력 시 실시간 영상 커버 이미지를 해상도별(1080p, 720p) 추출.
8. **색상 추출기 (`color-picker`)**: 레퍼런스 이미지에서 추출된 5대 대표 HEX/RGB 코드 팔레트 제공.
9. **QR 생성기 (`qr`)**: URL이나 텍스트를 QR코드 이미지로 변환 및 PNG 저장 기능 (실제 작동하는 QR API 연동).
10. **포맷 변환기 (`converter`)**: 이미지, 문서, 오디오 포맷의 가상 확장자 정렬 변환.
11. **메타데이터 추출기 (`metadata`)**: 파일 헤더에 기재된 EXIF 규격 정보 디코딩 표 제공.
12. **코드 뷰티파이어 (`code-beautifier`)**: 들여쓰기가 흐트러진 HTML, CSS, Javascript, JSON 텍스트 줄정돈 포매터.

---

## 3. UI 및 동적 라우팅 (Routing Structure)
* **메인 대시보드**: `/studio/tools/page.tsx`
  - 사용 가능한 도구 카테고리별 요약 위젯 및 12개 퀵 패널 링크 카드.
* **유틸 상세 보드**: `/studio/tools/[section]/page.tsx`
  - 동적 세그먼트 `section`에 따라 `src/app/studio/tools/[section]/components/`에서 해당 도구 컴포넌트를 마운트.
