# 유튜브 트렌드 스튜디오 설계 명세서 (YouTube Trend Studio Design Spec)

이 문서는 유튜브 트렌드 분석 스튜디오 모듈의 설계 의도, 보안 게이트웨이 파이프라인 및 데이터 장애 극복 정책을 기술한 기술 사양서입니다.

---

## 1. API 보안 및 게이트웨이 설계 (API Security & Gateway)
* **보안 통제**: Google YouTube Data API v3 키가 클라이언트 브라우저의 Network 탭에 그대로 노출되면 타인이 키를 탈취해 할당량(Quota)을 남용할 우려가 있습니다.
* **서버사이드 게이트웨이 패턴**: 클라이언트는 로컬 프록시 주소인 `/api/youtube?type=...` 로만 호출하며, 해당 API 라우트는 서버 백그라운드 환경에서 Supabase 데이터베이스 내 `admin_api_vault`에 저장된 암호화된 API 키를 로드 및 해독(`decryptApiKey`)하여 구글 엔드포인트와 통신합니다.

---

## 2. 장애극복 및 고가용성 정책 (High Availability & Fallback Simulation)
* **할당량 제한 극복**: 유튜브 무료 API 키는 일일 10,000 유닛의 엄격한 제한을 갖습니다. 트래픽 폭증으로 인한 `Quota Exceeded` 에러가 감지되거나 DB 키가 만료된 상황에 대비하여, 프록시가 자동으로 캐치하고 사실적인 메트릭(조회수, 구독자, 인기 비디오 리스트 등)을 담은 **Mock Fallback Payload**를 리턴합니다.
* **사용자 경험(UX) 보장**: API 실패 에러를 화면에 띄우는 대신 지연 없이 결과를 렌더링함으로써 사용자가 언제나 시각적 대시보드와 슬라이더 분석을 끊김 없이 체험할 수 있도록 하였습니다.

---

## 3. UI/UX 세부 디자인 규칙
* **모듈별 색상 체계**: 전체 메인 카드 상단 그라데이션은 뮤직 스튜디오 및 유튜브 본연의 아이덴티티인 레드-오렌지 무드(`from-zinc-900 to-[#1a0707]`)를 가미하여 일체감 있는 무드를 제공합니다.
* **비율 맞춤 차트**: 쇼츠 영상 시청 유지력 시뮬레이션 시, 고가의 외부 차트 라이브러리 설치로 인한 번들 크기 증가를 피하기 위해 SVG Path 좌표 연산 방식을 적용하여 가볍고 빠른 드로잉을 구현했습니다.
