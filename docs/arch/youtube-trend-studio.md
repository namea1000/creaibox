# 유튜브 트렌드 분석 스튜디오 (YouTube Trend Studio)

이 문서는 유튜브 트렌드 분석 스튜디오 모듈의 현재 아키텍처 및 API 연동에 대해 설명하는 상시 운영용 문서입니다.

---

## 1. 목적 (Purpose)
* 유튜브 채널 및 영상 크리에이터를 위한 종합적인 분석 도구입니다. 유튜브 Data API v3를 활용하여 실제 데이터(구독자, 누적 조회수, 인기 트렌드)를 가져오고, 제목 및 썸네일을 최적화하여 채널 성장을 유도합니다.

---

## 2. 주요 기능 (Main Features)
1. **채널 상세 분석 (`channel`)**: 채널 핸들 또는 이름 검색을 통해 실시간 구독자 수, 총 조회수, 비디오 개수와 최신 비디오 목록 스캔.
2. **급상승 영상 트렌드 (`rising`)**: 대한민국 급상승 인기 비디오 목록을 썸네일, 채널명, 실시간 조회수와 함께 실시간 스캔.
3. **경쟁 채널 비교 (`compare`)**: 두 채널의 주요 지표(구독자, 조회수, 업로드 비디오 수)를 1:1 비교하는 진단바 시각화.
4. **광고 단가 계산기 (`cpm`)**: CPM 단가, 조회수, 실제 광고 로드 비율 조정을 통한 예상 크리에이터 실수령액(RPM) 연산.
5. **유튜브 SEO 분석 (`seo`)**: 동영상 주소를 분석해 최적화 제목 글자수 검사, 설명 분량 및 등록된 메타 태그 목록 시각화.
6. **쇼츠 바이럴 분석 (`shorts`)**: 15s/30s/59s 시간대별 예상 시청 지속 지수(Retention Curve) 시뮬레이션 및 바이럴 핵심 팁 체크표.
7. **썸네일 CTR 연구소 (`thumbnail`)**: 유튜브 피드 내 경쟁사 썸네일 사이에서 시각적 가시성을 비교 분석하는 가상 피드 시뮬레이터.
8. **AI 제목 생성기 (`title`)**: 공용 Gemini API 및 로컬 백업 템플릿 엔진을 활용해 클릭율(CTR)이 높은 최적화 제목 5종 생성.
9. **콘텐츠 전략 리포트 (`report`)**: 틈새 채널 주제 및 업로드 주기를 입력하면 채널 포지셔닝안 및 4주차 스케줄 캘린더 리포트 구성.
10. **유튜브 자동 제작 연결 (`workflow`)**: 최적화 태그 분석과 시나리오 대본 구성을 일괄 완료한 후 비디오/음악 스튜디오로 다이렉트 연계.

---

## 3. UI 및 라우팅 구조 (UI Structure)
* **메인 제어 보드**: `/studio/youtube/page.tsx`
  - 누적 현황 카드, 10대 핵심 도구 퀵링크 카드 및 빠른 분석 태그 클라우드 배치.
* **상세 연동 도구**: `/studio/youtube/[section]/page.tsx`
  - URL 파라미터 세그먼트(`section`)에 맞추어 `src/app/studio/youtube/[section]/components/` 내부의 컴포넌트들을 조건부 렌더링.

---

## 4. 백엔드 API 프록시 구조 (API Gateway Integration)
* **보안 라우터**: `src/app/api/youtube/route.ts`
  - 클라이언트 브라우저에 구글 API 키 노출을 방지하기 위해 서버사이드에서 데이터베이스의 암호화된 키를 해독(`decryptApiKey`)하여 API 호출 수행.
  - 구글 서버의 쿼터 오버헤드 또는 API 키 만료 등의 예외 발생 시, 즉석에서 고품질 Mock 데이터를 자동 리턴하는 **자율 Fallback 구조** 장착.
