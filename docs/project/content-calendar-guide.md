# 크리에이박스(CreAibox) AI 콘텐츠 캘린더 마케팅 및 기능 명세서

이 문서는 크리에이박스(CreAibox) 플랫폼 내 **"AI 콘텐츠 캘린더"** 기능의 홍보 자료 제작, 마케팅 가치 제안(Value Proposition), 그리고 시스템 상세 스펙을 정리한 문서입니다.

---

## 1. 홍보용 핵심 셀링 포인트 (Selling Points)

### 📢 올인원 시각적 배포망 통제
* **한 페이지에서 끝나는 스케줄링**: 흩어져 있는 블로그, 네이버, SNS 포스팅 이력을 하나의 달력 안에서 일목요연하게 파악하여 채널 관리의 리소스를 극도로 줄여줍니다.
* **직관적인 다채널 필터링**: 기획 완료된 기획서 정보(수평 스케줄)와 실제 채널에 송출된 외부 배포 완료 정보가 서로 다른 컬러의 뱃지로 동적 매핑되어 퍼블리싱 상태를 즉시 분별할 수 있습니다.

### ⚡ 실시간 데이터베이스 연계
* **Supabase 기반 초정밀 상태 동기화**: 수동으로 일정을 기입할 필요 없이, 크리에이박스 스튜디오 내부에서 기획이 끝나거나 외부 API 배포가 성공하는 즉시 달력에 데이터가 자동 적재되어 실시간 발행율을 모니터링합니다.

### 🔗 원클릭 본문 링크 이동
* **배포지 즉시 추적**: 캘린더에 등재된 이벤트 카드 클릭 시 활성화되는 상세 모달에서 "발행 본문 보기" 단추를 누르면 외부 채널의 실제 배포 페이지로 1초 만에 다이렉트 랜딩됩니다.

## 2. 탑재된 주요 기능 스펙 (Feature Specs)

| 기능 분류 | 세부 사양 및 작동 메커니즘 |
| :--- | :--- |
| **반응형 월간 Grid 캘린더** | CSS Grid를 사용해 모던한 다크 테마 기반의 6행 7열 달력 뷰를 직접 설계하여, 외부 라이브러리 의존성 없이 유연하게 늘어나는 뷰포트를 제공합니다. |
| **월간 네비게이터** | 이전달, 다음달 및 오늘(Today) 이동 훅을 탑재하여 과거 데이터 추적과 미래 발행 계획 조회가 직관적입니다. |
| **이벤트 범례 및 실시간 카운트** | 상단에 기획 완료, 블로그 발행, 동영상/SNS 발행의 색상별 범례와 이번 달 총 생성 건수를 계량 지표로 표기해 마케팅 KPI 분석에 기여합니다. |
| **상세 정보 모달 팝업** | 일자별 콘텐츠 카드를 클릭하면 팝업 모달이 드러나 상세 제목, 기획 날짜, 배포 플랫폼, 메인 키워드 주제를 온전히 확인합니다. |
