# 지식 및 작가 페르소나 통합 매니저 개발 계획서

이 문서는 AI 글쓰기 구동 시 필요한 What(참조 지식 팩트 정보)과 How(작가 페르소나 어조/타겟) 설정을 한 공간에서 유기적으로 일체화하여 제어할 수 있는 **"지식 & 페르소나"** 통합 대시보드를 론칭하기 위해 수립되었던 개발 계획 명세서의 백업 보관 파일입니다.

---

## 1. 핵심 사양 및 리팩토링 설계 명세

### 1.1 사이드바 메뉴 격상 및 개명 (`Sidebar.tsx`)
* 기존 글쓰기 하위의 `지식 베이스` 메뉴 명칭을 **`지식 & 페르소나`**로 변경하여, 작가 프로필과 지식을 한곳에서 제어함을 직관적으로 나타냈습니다.

### 1.2 탭 기반의 지식 & 페르소나 통합 대시보드 (`knowledge/page.tsx`)
* **작가 페르소나 설정 탭 (탭 ①)**:
  * 작가명/필명, 전문 분야 카테고리, 대표 어조/말투(드롭다운), 타겟 독자층 및 상세 바이오 설명을 입력 및 수정할 수 있는 정교한 폼을 제공합니다.
  * 저장 시 브라우저 `localStorage` 에 `creaibox_persona_profile` 로 직렬화 저장되어 데이터 유실을 차단합니다.
  * 우측 어사이드 공간에 작성 정보가 실시간 반영되는 카드형 프리뷰를 배치해 소유감을 격상시킵니다.
* **참조 지식 아카이브 탭 (탭 ②)**:
  * 기존에 완비한 지식 카드별 글자 수/바이트 계량, 실시간 키워드 검색, 태그 필터 바 및 슬라이드오버 CRUD 모달 시스템을 두 번째 탭 하위로 온전히 이식했습니다.
