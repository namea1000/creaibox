# 영상 템플릿 마켓플레이스 & 레이어 교체 시스템 기획 설계서

본 문서는 크리에이박스(CreAibox)의 **영상 템플릿 마켓플레이스(Video Template Marketplace)**, **다중 비디오 합성 엔진(Multi-Video Merger)** 및 **사진 슬라이드쇼 전환 효과(Photo Slideshow Transitions) 컴파일러**의 상세 기획과 아키텍처 명세를 기술합니다.

---

## 1. 기획 및 설계 배경
기존의 단일 배경 비디오 교체 형태를 확장하여, 사용자가 여러 개의 영상 파일이나 사진들을 한데 불러와 간편하게 이어 붙이고, 부드러운 화면 전환 연출(Transition Effects)과 자막 오버레이를 적용해 **순식간에 완성도 높은 쇼츠/릴스 동영상을 컴파일**할 수 있도록 템플릿 편집 기능을 고도화했습니다.

---

## 2. 핵심 기능 아키텍처

```
┌────────────────────────────────────────────────────────┐
│             미디어 레이어 간편 편집실 (Modal)            │
└────────────────────────────────────────────────────────┘
     │                        │                    │
     ▼ (다중 비디오 순차 연결)   ▼ (사진 전환 오버레이)   ▼ (텍스트 렌더링)
┌─────────────────┐      ┌──────────────────┐ ┌──────────────────┐
│  Multi-Video    │      │ Photo Slideshow  │ │ Typography       │
│  Merger         │      │ Renderer         │ │ Subtitle Overlay │
└─────────────────┘      └──────────────────┘ └──────────────────┘
     │                        │                    │
     └────────────────────────┴──────────┬─────────┘
                                         │ (합성 및 오디오 믹싱)
                                         ▼
                             ┌───────────────────────┐
                             │  HTML5 Canvas 540x960 │
                             │  & MediaRecorder      │
                             └───────────────────────┘
```

### 1) 다중 비디오 순차 합성 (Multi-Video Merger)
* 사용자는 파일 탐색기를 통해 여러 개(`multiple`)의 동영상 클립을 동시에 지정해 올릴 수 있습니다.
* **비디오 분량 감지**: 업로드 시 브라우저 단에서 임시 비디오 객체를 생성하여 각 영상의 실제 재생 길이(Duration)를 초 단위로 정확하게 자동 추출합니다.
* **합성 루프**: Canvas 프레임 드로잉 시, 전체 타임라인 내에서 현재 프레임의 타임스탬프를 계산하여 활성화된 순서대로 동영상 트랙을 스위칭 및 탐색(Seeking)해가며 끊김 없이 이어 붙여 그립니다.

### 2) 사진 슬라이드쇼 & 전환 효과 (Photo Slideshow with Transitions)
* 여러 장의 사진을 템플릿에 추가하여 비디오 클립 뒤에 이어지는 슬라이드쇼를 생성할 수 있습니다.
* **장당 노출 시간 및 전환 효과 설정**:
  * **노출 시간**: 사진 1장당 재생 초 수(기본 3초)를 유저가 직접 입력할 수 있습니다.
  * **전환 효과 (Transitions)**:
    * **디졸브 (Fade)**: 이미지 교차 구간(0.5초) 동안 현재 이미지의 불투명도(`1 - progress`)와 다음 이미지의 불투명도(`progress`)를 믹싱하여 오버랩합니다.
    * **슬라이드 (Slide)**: 현재 이미지를 왼쪽으로 서서히 밀어내고(`-shiftX`) 동시에 오른쪽에서 다음 이미지가 밀려 들어오도록 가로축 좌표를 보간합니다.
    * **서서히 확대 (Zoom)**: 켄번(Ken Burns) 효과와 같이 슬라이드 시작 시점부터 끝날 때까지 스케일을 점진적으로 확대(`1.0` ➡️ `1.15`)하여 정적인 사진에 입체적인 운동감을 부여합니다.

### 3) 동적 타임라인 컴파일러
* 전체 비디오 재생 시간(Total Duration)을 다음과 같이 동적으로 자동 계산합니다:
  $$\text{총 분량} = \sum (\text{개별 비디오 분량}) + (\text{선택된 사진 장수} \times \text{장당 노출 초})$$
* Canvas에 비디오/사진/자막을 드로잉하는 동시에 선택된 템플릿의 배경 음악(BGM) 오디오 노드 데이터를 실시간 믹싱 녹음하여 하나의 통합 비디오 웹엠(`.webm`) 파일로 유저의 다운로드 디렉토리에 저장해 줍니다.

## 3. 사용자 인터페이스 (UI/UX) 구현 명세

1. **미디어 슬롯 (Media Slots)**:
   * **비디오 슬롯**: `내 컴퓨터에서 동영상 파일들 선택하기` 단추를 통해 여러 비디오를 올릴 수 있으며, 선택된 목록의 파일명과 정밀 재생 시간이 나열되고 개별 삭제(X)가 가능합니다.
   * **사진 슬롯**: `내 컴퓨터에서 사진 파일들 선택하기` 단추를 지원하며, 장당 노출 시간 조정 칸과 3가지 전환 효과(디졸브, 슬라이드, 줌) 드롭다운을 제공합니다. 추가된 사진은 귀여운 썸네일 그리드로 정렬됩니다.
2. **자막 슬롯 (Texts)**:
   * 중앙에 배치될 메인 볼드 타이틀 및 하단 서브 요약 문장을 즉석에서 변경할 수 있습니다.
3. **템플릿 비디오 정보 패널**:
   * 유저가 비디오나 사진을 추가/제거할 때마다 **"합산 예상 총 분량: X초"** 및 각 파트별 수량이 실시간 동적 텍스트로 업데이트되어 직관적인 피드백을 제공합니다.
