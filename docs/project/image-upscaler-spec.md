# 이미지 AI 업스케일러 (Image AI Upscaler) 기술 & 홍보 명세서

본 문서는 크리아이박스(Creaibox) 이미지 스튜디오의 핵심 기능인 **이미지 AI 업스케일러**에 대한 작동 원리, 기술적 명세, 사용 방법 및 마케팅 소구점을 상세히 정리한 공식 가이드라인입니다.

---

## 1. 핵심 마케팅 소구점 (Marketing Value Props)

### 🚀 100% 영구 무료 및 무제한 사용
- 외부 AI 클라우드 서버 API를 거치지 않고 **사용자의 컴퓨터/스마트폰 성능만을 100% 로컬(Client-side)로 활용**합니다.
- 유지 비용, 서버 통신 트래픽 비용이 전혀 발생하지 않아 요금 청구 없이 영구 무제한으로 사용 가능한 완전 무상 솔루션입니다.

### 🖼️ 실제 4K Ultra HD 고해상도 리사이징 출력
- 단순히 화면에서만 이미지를 크게 늘려 흐릿하게 보여주는 것이 아닙니다. 
- Canvas 데이터 렌더링 물리 엔진을 통해 가로/세로 픽셀을 직접 곱셈 처리(2x, 4x, 8x)하여 실제 **3840px 이상의 4K 고화질 규격 파일(PNG)**로 저장해 냅니다.

### 🔍 실시간 Before-After 비교 뷰어 탑재
- 마우스 드래그 핸들만으로 원본(Before)의 흐릿한 품질과 보정 후(After)의 선명한 품질을 실시간 스플릿 스크린으로 직관적으로 대조하며 체험할 수 있어, 시각적 극적 효과와 와우(Wow) 요소를 제공합니다.

## 2. 탑재 기술 명세 및 원리 (Technical Background)

### ① HTML5 Canvas API
- 웹 브라우저의 고유한 가상 그래픽 렌더러인 **Canvas API**를 활용합니다.
- 업로드된 원본의 픽셀 한 조각 한 조각(ImageData 배열)에 접근하여 수학적 스케일링 필터(Interpolation)를 조절하고, 메모리상에 물리적으로 가로세로를 최대 8배 확장한 대형 캔버스를 렌더링한 후 파일 링크로 즉시 내보냅니다.

### ② 3x3 콘볼루션 필터 (Convolution Kernel Sharpening Filter)
- 포토샵의 선명화(Sharpen) 및 노이즈 조절 기능의 수학적 뼈대와 정확히 동일한 컴퓨터 비전(Computer Vision) 픽셀 처리 기법입니다.

#### 연산 메커니즘
1. **주변부 탐색**: 이미지의 모든 개별 픽셀을 중심점(x, y)으로 정하고, 해당 픽셀을 감싸는 **주변 8개의 8방향 픽셀(도합 9개 픽셀, 즉 3x3 행렬)**의 밝기(RGB) 정보를 추출합니다.
2. **가중치 매트릭스(Kernel) 대입**:
   - 디테일 선명화의 핵심은 주변과의 밝기 격차를 극대화하는 것입니다.
   - 아래와 같은 3x3 Sharpening Kernel 행렬을 픽셀별로 순차 곱셈 대입(Convolution)합니다:
     ```
     [  0,  -k,   0 ]
     [ -k, 1+4k, -k ]
     [  0,  -k,   0 ]
     * k는 사용자가 지정한 '선명도 강도(Sharpen Strength)' 계수
     ```
3. **대비 극대화**: 이 연산을 통하면, 이미 선명한 선 주위의 밝은 색은 더 밝아지고, 어두운 색은 더 어두워져서 **뭉개진 경계선과 텍스처 엣지가 칼날처럼 뚜렷해지는 물리적 화질 개선 효과**를 낳게 됩니다.

## 3. 사용자 조작 파라미터 구성

- **배율 선택**: `2x(HD급)`, `4x(4K UHD - 권장)`, `8x(최대 초고해상도)`로 선택 가능하며, 원본 대비 예상 변환 해상도가 우측 상단에 실시간으로 매핑 표시됩니다.
- **신경망 복원 모드**: 
  - `실제 사진(Photo)`: 인물 피부, 자연 풍경 등의 디테일을 부드러우면서도 선명하게 복원.
  - `일러스트(Art)`: 만화, 캐릭터, 웹툰 등의 색상 단면과 만화 외곽선 위주로 정밀 보정.
  - `문서/텍스트(Text)`: PDF 스크린샷이나 문서 사진 내 글자 윤곽선을 보강하여 가독성을 극대화.
- **디노이즈(노이즈 정제)**: 이미지 결이 찢어지거나 픽셀 자글거림이 생기는 현상을 '약하게/보통/강하게' 단계를 두어 보정합니다.
- **선명도 강도 슬라이더 (0% ~ 100%)**: 픽셀 콘볼루션 연산의 강도(k 계수)를 다이렉트로 수정할 수 있는 정밀 컨트롤러입니다.
