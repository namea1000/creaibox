# CreAibox 개발일지 (Development Log Index)

이 문서는 `CreAibox` 프로젝트의 일자별 개발 내역 및 핵심 아키텍처 결정 사항(Devlog)을 월별로 관리하는 통합 인덱스 문서입니다.

---

## 📅 월별 개발일지 목록

과거의 개발 기록은 파일 용량 관리 및 로드 성능 최적화를 위해 월별 문서로 분할되어 정돈됩니다.

### 1. [2026년 7월 개발일지 (2026 July Devlog)](file:///Users/a1234/Local%20Sites/creaibox/docs/project/2026JulyDevlog.md)
* **주요 작업 내역**:
  * 블로그 하단 커스텀 에디토리얼 상자 UI 모달 제어 및 실시간 라이브 미리보기 구현.
  * 데이터베이스 테이블 확장 없는 HTML 주석 기반 JSON 데이터 직렬화 바인딩.
  * 블로그 및 브랜드 게시글 본문 크기(`text-[1.05rem]`)로 에디토리얼 글자 크기 통일.
  * 원고 목록 사이드바 접기/펼치기 제어 및 UI 가로 헤더선 1:1 수평 정렬 개편.
  * 사이드바 접기 시 텍스트 찌그러짐을 제거하는 고정폭 클리핑 마스크(Inner Wrapper) 도입.
  * 접혀 있을 때 노출되는 펼치기 아이콘 왼쪽에 직관적인 `목록` 안내 텍스트 추가.
  * 비디오 에디터 내 비디오 첫 프레임 자동 썸네일 표출 및 마우스 무한 60fps 스크러빙 탐색 탑재.

### 2. [2026년 6월 및 이전 개발일지 (2026 June Devlog)](file:///Users/a1234/Local%20Sites/creaibox/docs/project/2026JuneDevlog.md)
* **주요 작업 내역**:
  * 다중 서브도메인 브랜드 ID(1행 1도메인) 및 커스텀 도메인 DNS 승인 어드민 테이블 개편.
  * 구글 드라이브(20TB) 연동 이미지 업로드 및 Bypass CORS/CORP 고속 CDN(lh3) 주소 체계 구축.
  * Google Imagen 3 및 Veo 비디오 생성 API 결합.
  * Spotify 스타일 오디오 플레이어 및 R2 CDN 캐싱 우회 스트리밍 아키텍처 구현.
  * 무료 공유 에셋 라이브러리(Pixabay 스타일) 닉네임 연동 및 다운로드/좋아요/저장 토글.
  * 무한 스크롤 및 유튜브 급상승 트렌드 카테고리별 비동기 병렬 Failover 캐시 통합.
  * SSR 미들웨어 로그인 세션 유지 쿠키 청크 분실 이슈 정정 및 회원 멤버십 레벨 동적 세션 연동.
