# 크리에이박스(CreAibox) AI 자동화 워크플로우 마케팅 및 기능 명세서

이 문서는 크리에이박스(CreAibox) 플랫폼 내 **"AI 자동화 워크플로우"** 기능의 홍보 자료 제작, 마케팅 가치 제안(Value Proposition), 그리고 시스템 상세 스펙을 정리한 문서입니다.

---

## 1. 홍보용 핵심 셀링 포인트 (Selling Points)

### 📢 24시간 멈추지 않는 콘텐츠 무인 공장
* **완벽한 자동 퍼블리싱 파이프라인**: 단순한 AI 글쓰기를 넘어 '트리거 &rarr; 필터링 &rarr; AI 기획 &rarr; 플랫폼 배포'의 전 과정을 한 번의 트리거 설정으로 완전 무인화하여 1인 크리에이터 및 마케터의 운영 공수를 제로(0)에 가깝게 줄여줍니다.
* **트렌드 감지 즉시 포스팅**: 트렌드 지수가 85점을 넘어서는 실시간 급상승 키워드가 뜨는 순간, 워크플로우가 자동으로 작동하여 대중이 가장 원하는 시점에 최적화된 블로그 글을 자동 발행합니다.

### 🧠 인터랙티브 비주얼 노드 맵
* **개발자 없이 짜는 코딩 없는 워크플로우**: 어려운 스크립트 작성이나 외부 API 설정 없이, 직관적인 비주얼 플로우 맵을 통해 트리거 노드부터 발행 노드까지의 계통도를 한눈에 그리고 파악할 수 있습니다.

### 🧪 즉시 실행 시뮬레이터
* **가상 런타임 탑재**: 내가 설계한 자동화 라인이 정상 작동하는지 "테스트 작동 실행"을 클릭하면, 노드를 따라 초록색 빛이 차례대로 점멸하며 로딩 스피너와 함께 프로세스 성공 유무를 시뮬레이션해 줍니다.

## 2. 탑재된 주요 기능 스펙 (Feature Specs)

| 기능 분류 | 세부 사양 및 작동 메커니즘 |
| :--- | :--- |
| **자동화 파이프라인 매니저** | 여러 개의 워크플로우 규칙을 템플릿 카드로 보유하고, 토글 스위치를 통해 원클릭으로 활성/비활성화 모드를 즉각 전환합니다. |
| **비주얼 노드 체인 렌더러** | HTML/CSS 기반의 반응형 플로우 노드 레이아웃을 독자 설계하여, 노드의 상태(Idle, Running, Success, Failed)를 실시간 애니메이션 효과와 컬러 테두리로 렌더링합니다. |
| **우측 노드 옵션 편집 패널** | 특정 노드를 선택하면 우측 슬라이드 패널이 반응해 해당 노드의 세부 변수(수집 소스, 트리거 임계점, 어조, 타겟 블로그 등)를 직접 편집하고 메모리에 즉시 일시 저장합니다. |
| **성공 알림 토스트 피드백** | 시뮬레이션 및 토글 제어 시, 우측 하단에 세련된 다크 글래스모피즘 알림 패널이 팝업되어 규칙 저장 사항과 구동 경과를 요약 보고합니다. |
