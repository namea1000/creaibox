# 이미지 크기 조절기 기능 정의서 및 작동 가이드북

이 문서는 크리에이박스 이미지 스튜디오에 탑재된 **"이미지 크기 조절기"** 도구의 기능 구성, 8대 필터 가공의 Canvas 제어 기술 사양 및 마케팅 소구를 정리한 명세서입니다.

---

## 1. 서비스 개요

* **페이지 라우팅 경로**: `/studio/image/resizer`
* **사이드바 메뉴 위치**: 이미지 스튜디오 메뉴 그룹 &rarr; "이미지 배경 제거기" 바로 하단
* **주요 요약**: 사용자의 컴퓨터 브라우저상에서 직접 구동되어 외부 트래픽 요금 없이 무제한 무료로 가로세로 픽셀 조정, 이미지 자르기(크롭), 회전/반전, 포맷 변환, KB 용량 압축 및 도트/흑백 필터까지 지원하는 올인원 웹 그래픽 편집기입니다.

---

## 2. 8대 이미지 편집 도구 상세

| 기능 탭 | 설명 및 사용자 제어 옵션 | 작동 원리 (Canvas API 기술 사양) |
| :--- | :--- | :--- |
| **크기 (Resize)** | 폭(Width)/높이(Height) 직접 변경 및 가로세로 비율 유지 기능. | 새 임시 Canvas에 변경된 해상도로 사이즈를 생성하고 `drawImage`로 리샘플링 렌더. |
| **자르기 (Crop)** | 1:1, 4:3, 16:9 비율 고정 또는 자유 영역 자르기 박스 오버레이 제공. | 캔버스 좌표 비율로 환산 후 `drawImage(canvas, sx, sy, sw, sh, 0, 0, sw, sh)` 연산. |
| **반전 (Mirror)** | 가로 방향 좌우 반전 및 세로 방향 상하 반전. | `scale(-1, 1)` 또는 `scale(1, -1)` 변환 후 마이너스 오프셋 드로잉 처리. |
| **회전 (Rotate)** | 좌/우 90도 회전 및 캔버스 가로세로 축 크기 변경. | 축 중심점 평행 이동(`translate`), `rotate(rad)` 후 캔버스 width/height 치환. |
| **압축 (Compress)** | 자동 최적 품질 압축 및 원하는 목표 파일 크기(KB) 압축 지정. | `canvas.toDataURL(mime, quality)` 호출 시 퀄리티 값을 0.05~0.95 범위로 이진 검색. |
| **변환 (Convert)** | PNG, JPEG, WEBP 세 가지 대표 웹 포맷 확장자 변경. | 다운로드 인코딩 단계에서 MIME-Type 분기를 동적으로 할당해 PNG/WEBP/JPG 출력. |
| **픽셀화 (Pixelate)** | 2px ~ 50px 블록 크기에 맞춘 모자이크 도트 필터링. | `getImageData` 추출 후 블록 루프 내부 픽셀 평균 색상을 계산해 일괄 덮어쓰기. |
| **흑백 (Grayscale)** | 회색 조색조 필터 또는 Threshold 기준 흑백만(이진화) 필터. | 픽셀 RGB값을 명도 가중치($0.299R + 0.587G + 0.114B$) 공식으로 단일화 연산. |

---

## 3. 핵심 아키텍처 및 강점

### 3.1 100% 클라이언트 사이드 프로세싱 (Client-Side Rendering)
* **보안 안정성**: 모든 이미지와 기밀 시안이 외부 웹서버로 전송되거나 클라우드 임시 폴더에 저장되지 않아 완벽한 프라이버시 보호가 실현됩니다.
* **지연 없는 반응 속도**: 인터넷 업로드를 거치지 않아 용량이 큰 고해상도 DSLR 이미지도 즉각적으로 편집 완료됩니다.

### 3.2 25단계 Undo / Redo 편집 히스토리 탑재
* 브러시 스트로크 및 필터 적용 전 단계마다 이미지의 `ImageData` 객체를 히스토리 스택에 누적 보관합니다.
* 사용자가 실수했을 때 **`Ctrl + Z` (실행 취소)** 및 **`Ctrl + Y` (다시 실행)** 단축키를 입력해 자유롭게 상태 이동이 가능합니다.

## 4. 마케팅 및 브랜드 홍보 소구 가이드

SNS 마케팅, 카피라이팅, 사용자 가이드북 작성 시 아래 소구 포인트를 강조해 활용합니다.

1. **"화질 깨짐 없는 선명한 초고속 크기 조절기"**
   * 브라우저 Canvas의 자체 리샘플링 필터링으로 깨짐을 줄이면서 무료로 이미지 폭과 높이를 맞춰 줍니다.
2. **"목표 용량(KB) 입력만으로 자동 맞춤 압축"**
   * 이메일 첨부나 웹사이트 업로드 한계 용량(예: 500KB)에 맞춰 최적의 품질 비율로 알아서 축소 인코딩해 주는 자동 압축 기능의 쾌적함을 알립니다.
3. **"블로그, 썸네일, 쇼핑몰 상품 컷을 한 스페이스에서"**
   * 자르고, 돌리고, 흑백 느낌을 입히고, PNG 투명도를 유지하는 모든 작업을 하나의 편집기 안에서 해결할 수 있는 가성비를 소구합니다.
