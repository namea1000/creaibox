# 기능 개발 현황 (Roadmap & Feature Status)

현재 `CreAibox` 프로젝트에 이미 구현되어 프로덕션 수준으로 동작하는 주요 기능 카탈로그입니다. (추정 없음, 코드 검증 완료)

---

## 1. 구현 완료된 기능 (Functional Milestones)

### 1-1. 다중 블로그 에디터 워크플로우 (`/studio/writing/*`)
- **Universal Blog Editor**: 드래그 앤 드롭 이미지 삽입, 에디터 내용 실시간 임시 저장 및 Tiptap 기반 스타일링 툴바 지원.
- **네이버 포스팅 워크샵**: 
  - URL/텍스트를 긁어와 AI로 재구성하는 '글 재창조(Recreate)' 엔진.
  - 검색어 상위 트랙 분석 및 키워드 난이도 가이드라인(Naver Analysis Tower).
- **워드프레스 다이렉트 연동**:
  - API 자격 증명을 기반으로 한 다중 워드프레스 계정 등록.
  - 썸네일 생성 및 본문 일괄 전송(Publish) 처리.
- **크리에이박스 포스팅**:
  - 메타 설명, 캐노니컬 URL 조율, 포커스 키워드 매칭 상태 및 구글 SEO 점수 실시간 체크.

### 1-2. 타임라인 기반 비디오 에디터 (`/studio/video/editor`)
- **멀티 트랙 타임라인**: 자막 레이어, 텍스트 효과 레이어, 미디어 트랙, 배경 오디오 믹서를 타임 스케일에 따라 제어.
- **오디오 믹서**: 오디오 볼륨 조정, 재생 제어(Play, Pause, Seek).
- **FFmpeg 렌더링 엔진**: 브라우저 로컬 환경에서 FFmpeg WebM-to-MP4 및 MP4 인코딩을 수행하는 모듈 탑재 (`convertWebmToMp4.ts`).

### 1-3. 뮤직 가사 & 비주얼라이저 (`/studio/music/*`)
- **가사 기획/생성 (`planning` & `lyrics`)**: 음악의 세부 컨셉, 장르, 가사를 AI로 생성 및 가사 임시 저장소 연계.
- **오디오 비주얼라이저 (`visualizer`)**: 오디오 스펙트럼 시각화 캔버스 제공 및 비디오 익스포터 연동 (`exportMp4.ts`).

### 1-4. 암호화 기반 API 금고 (`/app/admin/apivault` & `/src/lib/server`)
- **AES-256-GCM 암호화**: API 키를 안전하게 데이터베이스에 난독화 보관.
- **API Failover 및 우선순위 분배**: 공용 키를 우선순위(`priority`), 일일 한도(`daily_limit`), 사용량(`today_count`) 순으로 선별하여 오류가 발생할 시 다음 활성 키로 자동 대체.
- **속도 제어**: 비회원/무료 회원용 일일 3회 제한(`FREE_DAILY_LIMIT = 3`)을 제어하는 로그 테이블 트래킹.

### 1-5. 통합 대시보드 및 통계 (`/app/admin/analytics`)
- **트래픽 모니터링**: 실시간 동접자, 오늘/어제/7일/30일 순 방문자 수 추이 산출.
- **다차원 지표**: 유입 채널별 비율 파이차트, 브라우저/디바이스/국가 통계 렌더링.
- **콘텐츠 생산 현황**: 작성완료/임시 저장 글 수 추적 및 전환율 모니터링.
